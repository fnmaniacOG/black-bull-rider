## Recording a clean gameplay clip

1. `npm run dev` → open http://localhost:3000 in Chrome (dev mode = free play, instant rounds, no wallet popups in the shot).
2. Zoom the page to ~110% (Cmd/Ctrl +) so the multiplier and letters fill the frame.
3. macOS: QuickTime → File → New Screen Recording → record just the browser window. Windows: Win+Alt+R (Game Bar). Or OBS with a 1920×1080 window capture.
4. Best clip structure (aim 25–35s):
   - 0–3s: "PAY 2 $ANSEM & RIDE" click (shows the burn text on screen)
   - 3–20s: frantic letter-whacking, a near-miss on a 💣, multiplier climbing
   - 20–25s: smash CASH OUT around 2.5–3x → green "CASHED OUT" screen
   - Bonus ending: one clip where you get BUCKED — losses are more shareable than wins
5. Crop out the browser chrome (or use a clean window, hide bookmarks bar: Cmd+Shift+B).
6. Speed any dead air 1.5x; add sound only if it's the game + a single bass hit at cashout.
7. For the real launch clip: run devnet mode instead and keep the Phantom approval popup IN the shot — the visible burn instruction is the proof that makes it credible.
